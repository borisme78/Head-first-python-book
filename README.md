## Ключові концепції Python

| Концепція | Опис |
|-----------|------|
| **Список (list)** | Впорядкована колекція, індексація з 0 |
| **Словник (dict)** | Колекція пар ключ-значення |
| **Множина (set)** | Колекція без дублікатів |
| **Рядок (str)** | Послідовність символів |
| **Модуль** | Бібліотека функцій (import) |
| **Цикл for** | Ітерація по колекції |
| **Умова if/else** | Блок коду залежно від умови |
